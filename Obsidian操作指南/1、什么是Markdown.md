# Markdown 与 Obsidian 知识科普

## 1. 什么是 Markdown？
**Markdown** 是一种轻量级的「标记语言」。它的核心哲学是：**用最简单的文本符号，实现最常用的排版效果。**

* **易读性：** 即使在没有渲染的纯文本状态下，内容也清晰易读。
* **兼容性：** 基于纯文本（.md），任何编辑器都能打开，永不过时。
* **专注力：** 写作时手不离键盘，无需频繁操作鼠标调整格式。

---

## 2. Obsidian 中的 Markdown
Obsidian 是一个以 Markdown 文件为基础的本地笔记软件。但在 Obsidian 里，Markdown 被赋予了 **“链接”** 和 **“数据库”** 的能力：

1.  **双向链接 (`[[ ]]`)：** Obsidian 的灵魂，让笔记不再是孤岛，而是互相连接的知识网络。
2.  **插件扩展：** 支持 Callouts（警告框）、Mermaid（流程图）等高级视觉组件。
3.  **Frontmatter (YAML)：** 在文件开头定义属性，让 Markdown 笔记变成可检索的数据库。

## 3. Markdown 常用语法清单

### 标题与文本
# 这是一级标题
## 这是二级标题
**这是加粗文本**，*这是斜体文本*，~~这是删除线~~。

### 列表
- 无序列表项 A
- 无序列表项 B
1. 有序列表第一项
2. 有序列表第二项

### 引用与代码
> 这是一段引用文字，通常用于摘抄。

`inline code`（行内代码）

### Obsidian 特色语法
- 双向链接：[[笔记标题]]
- 任务列表：
    - [ ] 未完成任务
    - [x] 已完成任务

### 数学公式 (LaTeX)
对于公式，使用 $ 符号包裹：
$$E = mc^2$$

### 表格
| 维度   | Markdown | 富文本(Word) |
| :--- | :------- | :-------- |
| 存储格式 | 纯文本      | 二进制/压缩包   |
| 编辑工具 | 任何编辑器    | 专用软件      |
| 掌握难度 | 极低       | 中等        |
